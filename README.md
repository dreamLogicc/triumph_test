# Тестовое задание ГК "Триумф"

## Инструкция по запуску
1. Клонируйте репозиторий 
    ```bash 
   git clone https://github.com/dreamLogicc/triumph_test.git
    ```
2. Установите зависимости
    ```bash
    pip install -r requirements.txt
   ```
3. Запустите
    ```bash
    python main.py
   ```
